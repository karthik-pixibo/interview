# League - API Assignment

##  About
Basic web server to perform matrix calculations

### Prerequisites
- git
- go

### Installation

```
$ git clone -b league https://github.com/karthik20522/interview.git
```

### Commands

-   Run test cases
```
$ go test .
```

-   Run
```
$ go run .
```

## Swagger Documentation

http://localhost:8080/swagger/index.html#/
